Política de Privacidade - Extensão Localizar Gerência

Última atualização: 21 de junho de 2025

A extensão "Localizar Gerência" é uma ferramenta desenvolvida para operadores de suporte técnico, projetada para facilitar o provisionamento e diagnóstico de ONUs em sistemas de fibra óptica, como AMS Nokia e SmartOLT. Esta Política de Privacidade explica como a extensão lida com dados e protege sua privacidade.

1. Dados Coletados

A extensão não coleta, armazena ou compartilha dados pessoais do usuário. Ela extrai informações públicas de páginas web visitadas, como território, cidade e empresa, para fins de consulta de equipamentos e provisionamento de ONUs. Esses dados são armazenados localmente no navegador (via chrome.storage.local) e não são enviados a servidores externos.

2. Uso dos Dados

Os dados extraídos (território, cidade, empresa) são usados exclusivamente para:

  • Identificar a gerência apropriada para provisionamento ou diagnóstico.
  • Consultar tabelas de equipamentos armazenadas localmente.
  • Gerar comandos para configuração de ONUs AMS Nokia.

A extensão também permite importar/exportar tabelas JSON localmente, sem envolvimento de servidores externos.

3. Recursos Externos

A extensão utiliza os seguintes recursos externos, acessados apenas por ação do usuário:

  • SmartOLT: Para configuração de ONUs.
  • Google Forms: Para relatar erros ou problemas.
  • Font Awesome (CDN): Para ícones de interface, sem acesso a dados do usuário.

4. Permissões

A extensão solicita permissões para:

  • activeTab: Acessar a aba ativa para extrair dados da página.
  • scripting: Injetar scripts para captura de dados e exibição de tabelas.
  • storage: Armazenar tabelas e dados capturados localmente.
  • tabs: Abrir URLs externas e interagir com abas para exibir tabelas.

Essas permissões são usadas apenas para as funcionalidades descritas.

5. Segurança

Os dados processados pela extensão permanecem no dispositivo do usuário e não são compartilhados com terceiros. A extensão segue as Políticas do Programa para Desenvolvedores da Chrome Web Store, garantindo segurança e privacidade.
